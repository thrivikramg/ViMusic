Description :

ViMusic is a simple Python script that allows you to download songs from a Spotify playlist using their names and search for them on YouTube. It then downloads the audio from the top search result on YouTube and saves it as an MP3 file.

Prerequisites
Before running the script, make sure you have the following installed:

Python 3
Spotipy
Pytube
Youtube Search Python
You can install these dependencies using pip:


Copy code
pip install spotipy pytube youtube-search-python

Usage
Run the script ViMusic.py.
Enter the link to the Spotify playlist when prompted.
The script will download the songs one by one from the playlist.

Note
Ensure that the Spotify playlist is public.
The downloaded files will be saved in the downloads directory.
